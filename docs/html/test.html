Linux & Bash Commandstest

test command is a command that is used to test the validity of a command. It checks whether the command/expression is true or false. It is used to check the type of file and the permissions related to a file. Test command returns 0 as a successful exit status if the command/expression is true, and returns 1 if the command/expression is false.
Syntax:
test [expression]
Examples:
test "variable1' operator "variable2"
