time is used to execute a command and prints a summary of real-time, user CPU time and system CPU time spent by executing a command when it terminates.
<s1>Syntax:</s1>
time [option] [COMMAND]
<img src="img/time (1).png" alt="Description of time">


<s1>Examples:</s1>
time -p :
This option is used to print time in POSIX format.
