case - Conditionally perform a command, case will selectively execute the command-list corresponding to the first pattern that matches word.

The case command in Linux is the best alternative when we had to use multiple if/elif on a single variable. It is used to execute the commands based on the pattern matching. 


<s1>Syntax</s1>

case WORD in [PATTERN [| PATTERN]...) COMMANDS ;;]... esac
