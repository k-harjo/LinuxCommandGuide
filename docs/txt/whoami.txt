whoami - Print the current user id and name. Same as id -un.


Syntax

      whoami [options]


Options

   --help      Display Help
   --version   Display program version info

The whoami utility has been obsoleted by the id(1) utility, and is equivalent to 'id -un'.
The command id -p is suggested for normal interactive use.

The whoami utility displays your effective user ID as a name.


*See also: id, who
