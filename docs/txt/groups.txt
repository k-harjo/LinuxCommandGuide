groups are the collection of users. Groups make it easy to manage users with the same security and access privileges. 

The `groups` is a powerful tool that helps administrators to view the group memberships of a user and retrieve information about user groups on a Linux System. 


Syntax  

groups [options] [username] 


Examples

Provided with a username: groups demon
